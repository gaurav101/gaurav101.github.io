LuminaJS

LuminaJS is a modular, chainable, lightweight, zero-dependency JavaScript utility library for browser-based image processing using the HTML5 Canvas API. It provides a functional approach to image manipulation, focusing on performance and ease of use.

Vanilla JS Demo React Demo Storybook(React) Documentation

Why LuminaJS?

Fast, small, and easy to integrate. LuminaJS works with HTML5 Canvas and supports chainable filters, React hooks, and direct script tag usage without any external dependencies.

  • 🚀 High Performance: optimized image processing loops
  • 🧩 Modular: import only the features you need
  • 📦 Lightweight: zero external dependencies
  • 🖼️ Canvas-powered: seamless browser integration

Quick Start

Install from npm and start processing images with a clean fluent API. 

npm install @gks101/luminajs

Use the `lumina()` chain to load images, apply filters, and render results directly to a canvas or HTML element.
